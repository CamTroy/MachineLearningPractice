# Goal
The goal of this lab is learn more about machine learning and compare ML models with the scikit-learn diabetes dataset.

# Running
To run this lab, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/SecondLab.git
   ```

2. Navigate to the project directory:
   ```
   cd MachineLearningPractice
   ```

3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

4. Run the Jupyter notebook:
   ```
   jupyter notebook
   ```
